JavaScript (Three.js) 크기 조정 확인 CSS 수정으로 `#threejs-container`가 동적으로 높이를 가져가도록 했으므로, JavaScript 내부에서 `renderer.setSize()`와 `camera.aspect`를 **실제 컨테이너의 크기**로 정확하게 업데이트해야 합니다. 제공해주신 JavaScript 코드에는 이미 `window.addEventListener('resize', ...)` 이벤트 핸들러가 포함되어 있어, 브라우저 창 크기가 변경될 때 Three.js 뷰어 크기를 업데이트합니다. ```javascript // JS 코드에서 이미 이 부분이 잘 동작하고 있습니다. window.addEventListener('resize', () => { camera.aspect = threeJsContainer.clientWidth / threeJsContainer.clientHeight; camera.updateProjectionMatrix(); renderer.setSize(threeJsContainer.clientWidth, threeJsContainer.clientHeight); });
